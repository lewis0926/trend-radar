Trend Radar is an assistant app that helps identify which industries are trending based on past price actions.

Project:
Send email notifications to users regularly. E.g. weekly
We'll have a list of ETFs/ indices that we want to track. They should have exposure to all kinds of industries in the market, including but not limited to tech, healthcare, finance, energy, consumer discretionary, etc.

Tech Stack:
- Python 3.10+
- venvs for dependency management
- Resend for email notifications
- Pkl for configs management. Generate json config from pkl file and parse it as configs in the app.
